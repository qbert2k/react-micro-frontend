# product

## Create Project

```shell
npx create-mf-app
```

- Pick the name of your app: “product” // name of the app
- Project Type: Application
- Port number: 9001
- Framework: React
- Language: JavaScript
- CSS: Tailwind

## Project Setup

```shell
npm install
npm i axios react-router-dom
```

## Run Project

```shell
npm start
```
